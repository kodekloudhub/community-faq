# 100 Days Challenges

This mini FAQ pertains to the the KKE challenges - 100 days of DevOps and 100 days of Cloud

## Task Unlock Logic

The following describes in what order and how frequently you can attempt the tasks, which is dependent on your subscription plan. Note that these specific rules *do not apply* to standard KKE practice where the rules are different, only to 100 Days challenges.

### Free and Standard Plans

**Unlock Logic**
* When a topic is added to **Active Practice, 1 task unlocks per day**
* Tasks unlock **strictly in sequence**
* Next task unlocks **only after the previous task is marked as completed**

**Attempt Restrictions**
* You can attempt **only 1 task per day**
* Even if the next task is unlocked, you must **wait until the next day** to attempt it
* **No skipping** of tasks allowed

### Pro and AI Plans

**Unlock Logic**
* When a topic is added to **Active Practice**, you can unlock **multiple tasks on the same day**
* Tasks unlock **strictly in sequence**
* Next task unlocks **only after the previous task is marked as completed**

**Attempt Restrictions**
* You can attempt** unlimited tasks per day**
* **No skipping** of tasks allowed
